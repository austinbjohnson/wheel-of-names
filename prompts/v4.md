# Wheel of Names — Liquid Glass / Zapier Edition

**Rebuild https://wheelofnames.com/ but make it like "liquid glass" design and styled to the Zapier brand. The prefilled people should be Enterprise Zone PMs.**

**Aesthetic direction:**
*   **Liquid glass** — translucent panels, backdrop blur, frosted refractive surfaces, chromatic edges, glossy highlights. Apple's recent visionOS / iOS direction is fair inspiration; don't copy it literally.
*   **Zapier brand** — Zapier orange (`#FF4A00` / `#FF4F00`) as the dominant accent. Warm palette around it (amber, peach, cream). Avoid generic SaaS gradients (purple-on-white, etc).
*   Find your own typography — don't default to Inter / Roboto / system stacks.

**Prefilled roster (Enterprise Zone PMs):**
The participants list should be **prefilled** with the Zapier Enterprise Zone Product Managers. The model is expected to figure out a reasonable list from available context (e.g. a working knowledge base, repo, or web search). Names should be editable / removable inline.

**Core requirements:**
*   Render a spinning wheel with names in pie slices.
*   Spin animation that decelerates naturally; pointer that clearly indicates the winner.
*   **Outcome must match the pointer** — the announced winner is the slice physically under the pointer.
*   Add / edit / remove names without reloading.
*   Winner announcement with a celebratory moment (confetti, banner, etc).
*   Recent-winner history is nice-to-have.

**Out of scope (vs. v3):**
*   No requirement for "Keep / Disable" post-spin dialog.
*   No dark mode requirement (the aesthetic is light + glassy by default).
